# Inventory Management System (IMS)

## Description
A desktop-based Inventory Management System developed using Python and MySQL with a GUI built using CustomTkinter.  
The system manages employees, suppliers, products, and billing operations.

## Tech Stack
- Python
- MySQL
- CustomTkinter

## Features
- Login Authentication
- Employee Management
- Supplier Management
- Product Management
- Billing & Invoice Generation
- Stock Update After Purchase

## Project Type
Academic / Self-learning Project
